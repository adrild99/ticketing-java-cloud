# Plataforma de Ticketing

El objetivo inicial es desarrollar la lógica de negocio, pero el proyecto está diseñado para ir escalando e incorporar persistencia de datos y una interfaz gráfica.

## Fases del Proyecto

- [x] **Fase 1:** Desarrollo de la lógica y estructura orientada a objetos en Java puro.
- [ ] **Fase 2:** Integración con Base de Datos Relacional para la persistencia de eventos, usuarios y tickets.
- [ ] **Fase 3:** Creación de GUI.  

## Tecnologías utilizadas

**Lenguaje:** Java
**Control de versiones:** Git / GitHub
